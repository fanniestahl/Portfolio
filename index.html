2D-Game

We were given the task to create a 2D game on a HTML site. We made a game based off of "Space shooter" but with our own twist. We named it "Monkey Business". This is my second game made in JS and the improvement is significant from the last game I made. The goal of the game is destroy the coconuts with bananas and reaching the score of 1000.
